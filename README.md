# October CMS Webpack starter

### Includes:

- Webpack (Laravel Mix + browserSync)
- Bootstrap
- jQuery
- Font Awesome

### How to:

- Install October CMS
- Create a new theme in October /themes folder
- Clone the repo to the theme folder
- Run `npm install` 
- Add own packages
- Run `npm run dev` or `npm run watch`
- Customize your new there

### Production:

- Run `npm run prod`
